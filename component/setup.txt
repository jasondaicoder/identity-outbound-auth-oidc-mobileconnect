Product: WSO2 IS authenticator for Mobile Connect
Pre-requisites:

- Maven 3.x
- Java 1.7 or above

Tested Platform: 

- Mac OSx 10.9
- WSO2 IS 5.1.0
- Java 1.7

Do the following:

Deploying and Configuring Mobile Connect artifacts:

1.  Build the <MOBILECONNECT_AUTHENTICATOR_HOME>/component/authenticator & copy the org.wso2.carbon.extension.identity.authenticator.mobileconnect.connector-1.0.5.jar
    to <IS-HOME>/repository/components/dropins

2.  Follow the steps in https://docs.wso2.com/display/ISCONNECTORS/Configuring+Mobileconnect+Authenticator
